# bookstore-back

# Backend do projeto Bookstore

# github links

# https
https://github.com/encode75/bookstore-back.git

# github cli
gh repo clone encode75/bookstore-back

# instalando dependências
npm install

# MongoDb
Faz-se necessária a instalação do MongoDB localmente ou usar uma URI de conexão no arquivo .env
Ex.: MONGO_URI="mongodb://localhost:27017/bookstore"

# rodar o servidor
npm run start
